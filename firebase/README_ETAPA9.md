# Etapa 9 — Reglas mínimas para miembros reales

En Firebase Console > Firestore Database > Reglas, copia el contenido de `firestore.rules` y publícalo.

Esto habilita el flujo de:
- espacio propio inicial,
- unión por código,
- cambio real de rol,
- remoción operativa,
- protección básica para que no cualquiera administre miembros.
