# Repo higiene — Suite A33
No subir carpetas __MACOSX/ ni archivos .DS_Store o ._* (AppleDouble).
Si aparecen, bórralos antes de commitear o crear ZIPs.
Evitar nombres largos/raros (sobre todo en manuales y exports).
Usa nombres cortos, sin acentos, sin caracteres especiales y sin rutas profundas.
Preferir kebab-case: manual-usuario-suite-a33-v4.20.77.pdf
Mantén manuales/exports con nombres limpios y consistentes.
No dupliques carpetas por guiones/underscore: respeta el canon del proyecto.
Antes de subir: revisa que el ZIP no traiga archivos ocultos del sistema.
Si Git se queja por el nombre, renómbralo a algo más corto y listo.
