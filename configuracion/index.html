    Respaldo

    JSON local

    Exporta o importa respaldos completos de la Suite A33. También puedes generar e importar JSON personalizados por módulos, submódulos y eventos POS sin borrar datos no incluidos.

    Qué hace

    Exporta un archivo completo o personalizado con localStorage e IndexedDB de la Suite detectados en este navegador.

    Qué no cambia

    El respaldo completo conserva su formato histórico; el personalizado se identifica como parcial.

    Cuidado

    Al importar un respaldo completo se reemplaza todo; al importar uno parcial se fusionan solo las secciones incluidas. Mejor exportar antes; drama evitado.

    JSON importados

    Registro compacto de los últimos respaldos importados en este navegador.

    Respaldo

    Operación segura

    Operativo
    • Exporta respaldo JSON completo.
    • Exporta JSON personalizado por módulos y submódulos.
    • Importa respaldos completos o parciales desde este navegador.
    • Mantiene el formato histórico del archivo completo.
    • Conserva lectura clara en desktop e iPad.
