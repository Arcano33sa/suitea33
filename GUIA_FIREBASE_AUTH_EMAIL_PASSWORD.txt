Firebase Console — habilitar Email/Password
1) Abre tu proyecto en Firebase Console.
2) Menú izquierdo → Compilación / Build → Authentication.
3) Abre la pestaña Sign-in method.
4) En proveedores, entra a Email/Password.
5) Activa la opción Email/Password.
6) Guarda con Save.
7) Opcional: pestaña Users → Add user para crear el primer acceso.
